认识期权（一）期权的基本概念

发表于2022-07-28|金融证券知识
|字数总计:869|阅读时长:2分钟

一、什么是期权

期权，是指一种合约，该合约赋予持有人在某一特定日期或该日之前的任何时间以固定价格购进或售出一种资产的权利。期权是一种权利。持有人享有权利但不承担相应的义务。

1.1 一个简单的例子

Apple公司预计2022年9月10日发行最新款的iPhone14，而就在此时一些零售商推出了“6000元换购iPhone14”的换购券，每张售价50元。这个例子中，购买了该购物券，便是享有了一项权利，即到2022年9月10日当天，可以自行决定是否以6000元的价格换购iPhone14，本质上享有的是一种自主权利，这便是期权的雏形。

1.2 几个基本要素

还是以上面这个例子，以期权的视角来看看对应哪些基本要素：

要素 对应内容
合约标的资产 iPhone14
合约单位 1台
行权价 6000元
到期日 2022年9月10日
交割方式 实物交割

到目前为止，大部分应该容易被理解。除了行权价这个概念，这里稍后再作解释。继续回到现实角度来看，什么情况下选择行使换购券权利更划算：

iPhone14官方发行价 直接购买的成本 使用换购券的成本 使用换购券收益
6500 6500 6000 + 50 = 6050 6500 - 6050 = 550
6100 6100 6000 + 50 = 6050 6100 - 6050 = 50
6050 6050 6000 + 50 = 6050 6050 - 6050 = 0
6000 6000 6000 + 50 = 6050 6000 - 6050 = -50
5500 5500 6000 + 50 = 6050 5500 - 6050 = -550

不难分析，只有当iPhone14的发行价 > 6050，使用换购券才划算。而随着发行价越高，使用换购券的收益率越高，换句话说就是越贵越划算。

1.3 从股票的视角来看

iPhone14发行后在很长一个周期内价格是不会变动的，而在发行前这段时间也不会有明确的价格体现。接下来我们试图把标的换成股票，看看又有哪些变化？

假设以港股 00700腾讯控股 正股为例，假设今日的收盘价为320港元。如果看好它，认为它在一段时间后会涨到350元，有几种投资策略？最简单的一种就是直接买入正股，除此之外还有吗？没错，就是我们谈到的期权。首先先找到一只腾讯期权来分析：

先解释下上图的几个基本要素含义：

基本要素 数值 具体含义
标的资产 腾讯控股正股 期权合约对应的标的证券
权利金 4.190 简单理解就是买这张期权的单位成本
行权价 350.000 标的资产到达这个价格后，可以行使期权权利
到期日 2022.08.30 该期权的有效时间
合约乘数 100 简单理解就是买这张期权的单位数量，期权总成本 = 权利金 * 合约乘数

购买该期权的成本为 4.190 * 100 = 419.00 。而买完之后享有什么权利呢？

持有人可以在2022.08.30当日收盘前，以350单价买入100股腾讯控股正股。 这里可能有2个疑问：

  1. 为什么是2022.08.30当日收盘前，而不是2022.08.30当日？
  2. 为什么是100股？

先留着这些疑问继续往下看。
